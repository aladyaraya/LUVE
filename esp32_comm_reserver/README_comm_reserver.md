# 跳蛋振动控制器

> **从机端 - BLE 客户端振动执行器**

---

## 📋 项目简介

本项目是双端联动系统的**从机端**，负责接收主机指令并执行振动控制。运行在 ESP32 开发板上，通过 BLE 连接主机，实时响应 `LEVEL:x` 指令调整振动强度。

**核心定位**：执行终端，动作响应者

---

## 🎯 核心功能

| 功能模块 | 描述 | 技术实现 |
|---------|------|---------|
| BLE 扫描 | 自动搜索主机设备 | BLE Client 模式 |
| 指令接收 | 订阅主机通知 | BLE Notify |
| 指令解析 | 解析 `LEVEL:x` 格式 | 字符串处理 |
| 振动控制 | 4 级 PWM 强度调节 | ledc PWM |
| 断线重连 | 自动检测并重连 | 连接状态监控 |

## 🔌 接线图

### 振动马达驱动


```
ESP32                    振动马达
────────                 ────────
GPIO 4 ───┬─ 1kΩ电阻 ───┤ G (栅极)
          │
         GND
          
MOS 管源极 (S) ────────→ GND
MOS 管漏极 (D) ──┬─────→ 马达负极
                 │
               二极管 (反向并联保护)
                 │
马达正极 ────────┴─────→ 3.7V/5V 电源
```

## ⚙️ 配置参数

### BLE 配置（通常无需修改）

```cpp
#define TARGET_DEVICE_NAME    "ESP32_FusionCtrl"  // 主机设备名
#define SERVICE_UUID          "4fafc201-1fb5-459e-8fcc-c5c9c331914b"
#define CHARACTERISTIC_UUID   "beb5483e-36e1-4688-b7f5-ea07361b26a8"
```

### 振动强度配置（可根据手感调整）

```cpp
#define PWM_FREQ    5000    // PWM 频率 (Hz)
#define PWM_RES     8       // PWM 分辨率 (位)

// 占空比 (0-255)
#define DUTY_OFF    30      // 最低档 (12%) - 几乎停止
#define DUTY_LOW    80      // 中低档 (31%) - 轻微震动
#define DUTY_MED    150     // 中高档 (59%) - 明显震动
#define DUTY_HIGH   250     // 最高档 (98%) - 强烈震动
```

## 🚀 快速开始

### 步骤 1: 硬件连接

1. 按照接线图连接振动马达驱动电路
2. 确认 GPIO 4 输出 PWM 信号
3. 检查电源供电能力（马达需要较大电流）

### 步骤 2: 开发板配置

```
开发板: ESP32 Dev Module
上传速度: 921600
CPU 频率: 240MHz
```

### 步骤 3: 上传代码

1. 打开 `esp32_comm_reserver.ino`
2. 点击上传
3. 打开串口监视器 (115200 波特率)

### 步骤 4: 启动测试

**自检流程**：
1. 上电后马达振动 2 秒（自检）
2. 开始扫描 BLE 设备
3. 找到 `ESP32_FusionCtrl` 后自动连接

## 📊 振动档位映射

### 档位对照表

| 接收指令 | 档位 | PWM 占空比 | 实际强度 | 说明 |
|---------|-----|-----------|---------|------|
| `LEVEL:0` | 最高档 | 250/255 (98%) | ████████ | 强烈震动 |
| `LEVEL:1` | 中高档 | 150/255 (59%) | █████░░░ | 明显震动 |
| `LEVEL:2` | 中低档 | 80/255 (31%) | ███░░░░░ | 轻微震动 |
| `LEVEL:3` | 最低档 | 30/255 (12%) | ░░░░░░░░ | 几乎停止 |

### 强度调整建议

如果振动强度不符合预期，修改以下参数：

```cpp
// 震动太弱 → 增大占空比
#define DUTY_HIGH   250  // 改为 255 (100%)

// 震动太强 → 减小占空比
#define DUTY_MED    150  // 改为 120 (47%)

// 最低档仍有明显震动 → 减小 DUTY_OFF
#define DUTY_OFF    30   // 改为 0 (完全关闭)
```
