# ESP32 统一传感器控制系统

> **主机端 - BLE + WiFi 双通道智能控制中枢**

---

## 📋 项目简介

本项目是双端联动系统的**主机端**，负责传感器数据采集、融合决策、双通道数据下发。运行在 ESP32-S3 开发板上，通过 BLE 实时控制从机振动，通过 WiFi 同步数据至 Web 服务器。

**核心定位**：系统大脑，决策中心

---

## 🎯 核心功能

| 功能模块 | 描述 | 技术实现 |
|---------|------|---------|
| 心率检测 | 实时监测心率 BPM | MAX30105 光电传感器 + I2C |
| 压力检测 | 检测按压强度 | FSR402 薄膜传感器 + ADC |
| 数据融合 | 压力优先 + 心率辅助 | 自定义决策算法 |
| BLE 服务端 | 向从机发送振动指令 | BLE 5.0 Server |
| WiFi 上传 | 定时推送数据至服务器web端 | HTTP POST |

## ⚙️ 配置参数

### 必须修改的配置

```cpp
// ================= WiFi 配置 =================
const char* WIFI_SSID = "你的WiFi名称";        // ← 修改为实际 SSID
const char* WIFI_PASSWORD = "你的WiFi密码";     // ← 修改为实际密码
const char* SERVER_URL = "http://192.168.x.x:5001/api/data";  // ← 修改为服务器地址
```

### 可选调整的配置

```cpp
// ================= 压力阈值 =================
#define PRESSURE_L0 500     // 轻压 → 最高档
#define PRESSURE_L1 1500    // 中压 → 中高档
#define PRESSURE_L2 3000    // 重压 → 中低档
// 超过 PRESSURE_L2 → 最低档/停止

// ================= 心率阈值 =================
#define HEART_RATE_L1 60    // < 60 BPM → 最低档
#define HEART_RATE_L2 80    // 60-80 BPM → 中低档
#define HEART_RATE_L3 100   // 80-100 BPM → 中高档
// > 100 BPM → 最高档

// ================= 滤波参数 =================
#define EMA_ALPHA 0.3f      // 压力滤波系数 (0-1)
#define SAMPLE_MS 20        // 采样周期 (毫秒)
```

## 🚀 快速开始

### 步骤 1: 安装依赖库

在 Arduino IDE 中搜索并安装：

```
库管理器 → 搜索 "MAX30105" → 安装 SparkFun MAX30105 库
库管理器 → 搜索 "heartRate" → 安装 SparkFun Heart Rate 库
```

### 步骤 2: 开发板配置

```
开发板: ESP32S3 Dev Module
上传速度: 921600
CPU 频率: 240MHz
Flash 模式: QIO
Flash 大小: 16MB
```

### 步骤 3: 修改配置

打开 `esp32_unified_system.ino`，修改 WiFi 配置：

```cpp
const char* WIFI_SSID = "你的实际WiFi名称";
const char* WIFI_PASSWORD = "你的实际密码";
const char* SERVER_URL = "http://你的服务器IP:5001/api/data";
```

### 步骤 4: 上传并测试

1. 点击 Arduino IDE 的上传按钮
2. 打开串口监视器 (115200 波特率)
3. 观察启动日志确认各模块初始化成功

## 📊 振动档位逻辑

### 档位定义

| 档位 | 强度 | 触发条件 | 说明 |
|-----|------|---------|------|
| 0 | 最高 | 轻压 或 高心率(>100) | 最强振动 |
| 1 | 中高 | 中压 或 中高心率(80-100) | 较强振动 |
| 2 | 中低 | 重压 或 中低心率(60-80) | 较弱振动 |
| 3 | 最低 | 极重压 或 低心率(<60) | 停止 |

### 融合决策算法

```cpp
uint8_t computeFinalLevel() {
  if (pressureLevel <= 2) {
    return pressureLevel;  // 压力优先
  } else {
    return mapHeartRateToLevel(beatsPerMinute);  // 心率辅助
  }
}
```

**设计理念**：

- 压力传感器作为**主动触发源**：用户按压时立即响应
- 心率传感器作为**被动调节器**：无按压时根据生理状态自动调节
